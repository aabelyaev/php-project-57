## About

This is a minimalistic CRM system for managing the flow of tasks and personnel.

## Requirements

* PHP ^8.2
* Node ^20.19.0 || >=22.12.0
* npm

## Setup

1. Download this project:

    ```shell

    cd php-project-57

2. Quick Start

    ```shell
    make setup-start
